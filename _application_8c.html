Main Application Module.

This file contains the main application code for PREN T15 robot. On startup of the microcontroller the APP_Init() and APP_Loop() are called from ProcessorExpert.c. Here also the Event Handler is implemented which does different actions depending on the generated event. Every part of the software can be the source of an event.
