Detailed Description

Serial Module implementation.

Author:
Christoph BÃ¤chler
Date:
12.11.2013

This module implements the serial protocol as described in the PREN1 design concept documentation. Core of the module is the SER_Process() function which implements a finite state machine to read the packets char-by-char from the serial interface.
